11. Set Theory¶

Set theory is a branch of mathematics that studies collections of objects known as Sets. The language of set theory, which will start to study here, can be used to define almost all mathematical objects (if you are interested in what cannot be so easily defined, look up ideas such as Gödel’s incompleteness theorem and Russell’s Paradox).

11.2. Definition of a set¶

There are two basic ways to define a set:

  • List / Tabular Form, where we list the members of the set in any order. For example, the set of all the vowels and consonants:

\[\begin{split} V &= \{a,\,e,\,i\,o,\,u\} \\ C &= \{b,\,c,\,d,\,f,\,g,\,h,\,j,\,k,\,l,\,m,\,n,\,p,\,q,\,r,\,s,\,t,\,v,\,w,\,x,\,y,\,z\}\end{split}\]

or the set of the Natural number and Integers:

\[\begin{split} \mathbb{N} &= \{1,\,2,\,3,\dots \} \\ \mathbb{Z} &= \{ \dots,\,-3,\,-2,\,-1,\,0,\,1,\,2,\,3,\,\dots \}\end{split}\]

  • Set-Builder Form / Property Method, where we state the properties which characterise the elements of a set, i.e. those held by members of the set (but not by non-members. For example:

\[ A = \{x\,:\, x \text{ is an integer}\}\]

which we read is “The set \(A\), defined by elements \(x\), such that \(x\) is an integer”. There are many ways to write out such a statement, the simplest is to use the wider notation of set theory. For example, the set of Rational numbers or Complex numbers:

\[\begin{split} \mathbb{Q} &= \left\{ \frac{m}{n} \,\,\middle|\,\, m,\,n \in \mathbb{Z}\right\} \\ \mathbb{C} &= \left\{ a + i\,b \,\,\middle|\,\, a,\,b\in \mathbb{R},\, i^2 = -1\right\}\end{split}\]

11.3. Venn Diagrams¶

We can represent sets pictorially using Venn diagrams (named after a Logician and Philosopher Dr John Venn, who’s grandfather Rev John Venn has a street named after him in Clapham, South London!), which are an easy to see how different sets can have shared and unique elements as well as defining the Universal set \(\mathbb{U}\) in a given context.

Considering our example of sets \(A,\,B,\,\mathbb{U}\):

\[\begin{split}\mathbb{U} &= \{ 1,\,2,\,3,\,4,\,5,\,6,\,7\}\\ A &= \{1,\,2,\,3,\,4\}\\ B &= \{3,\,4,\,5\}\end{split}\]

this could be represented by the Venn diagram in Fig. 11.1.

Fig. 11.1 Venn diagram depicting sets \(A,\,B,\,\mathbb{U}\).¶

We can see that the union of both sets is given by the entire shaded area

\[A \cup B = \{1,\,2,\,3,\,4,\,5\}\]

whilst the intersection of both sets is given by the shared shaded area

\[A \cap B = \{ 3,\,4\}\]

For the difference between the sets,

\[\begin{split}(A\cup B) \setminus B &= \{ 1,\,2\} \\ (A\cup B) \setminus A &= \{ 5\} \end{split}\]

these could be represented by Venn diagrams, such as that seen in Fig. 11.2.

Fig. 11.2 Venn diagram depicting sets \(A\), \(B\), \(A \cup A\), \(| \cap B\) and \((A\cup B) \setminus B\).¶

and likewise the complement

\[\begin{split}\neg A &= \{5,\,6,\,7\} \\ \neg B &= \{1,\,2,\,6,\,7\} \\ \neg (A \cup B) &= \{6,\,7\}\\ \neg (A \cap B) &= \{1,\,2,\,5,\,6,\,7\} \end{split}\]

these could be represented by Venn diagrams with appropriate shading.

11.4. Laws of the Algebra of Sets¶

Sets follow a specific algebra, which we can deduce by following the logical principles outlined previously. There are identity laws, i.e. can we use AND or OR like multiplying or adding on 1 or 0:

\[\begin{split}A \cup \varnothing &= A \\ A \cap \varnothing &= \varnothing \\ A \cap \mathbb{U} &= A \\ A \cup \mathbb{U} &= \mathbb{U}\end{split}\]

Associative laws, i.e. does the position of the brackets matter:

\[\begin{split}(A \cup B) \cup C &= A \cup (B \cup C)\\ (A \cap B) \cap C &= A \cap (B \cap C)\end{split}\]

Commutative laws, i.e. does the order of operations matter:

\[\begin{split}A \cup B &= B \cup A \\ A \cap B &= B \cap A\end{split}\]

Distributive laws, i.e. does the operation in a bracket expand out:

\[\begin{split}A \cup (B \cap C) &= (A\cup B) \cap (A \cup C)\\ A \cap (B \cup C) &= (A\cap B) \cup (A \cap C)\end{split}\]

Involution laws, i.e. is the inverse of the inverse return to the original state:

\[\neg(\neg A) = A\]

Idempotent laws, i.e. it’s everything or nothing:

\[\begin{split}A \cup A &= A \\ A \cap A &= A\end{split}\]

Complement laws, i.e. does everything add up together or subtract to zero:

\[\begin{split}A \cup \neg A &= \mathbb{U} \\ A \cap \neg A &= \varnothing \\ \neg \mathbb{U} &= \varnothing \\ \neg \varnothing &= \mathbb{U}\end{split}\]

And finally DeMorgan’s laws, which refer to overall inverse relations:

\[\begin{split}\neg(A \cup B) &= \neg A \cap \neg B \\ \neg(A \cap B) &= \neg A \cup \neg B\end{split}\]

11.5. Counting Sets¶

It is often useful to count elements within sets and therefore use the logical extensions of these ideas to find the particular areas of shaded Venn diagrams.
If \(A\) and \(B\) are finite, disjoint sets, i.e. \(A \cup B = C\) is finite and \(A \cap B = \varnothing\), then

\[n(C) = n(A) + n(B)\]

A simple example of this is

\[n(\mathbb{U}) = n(A) + n(\neg A)\]

which is simply joining up a set and its complement. One issue with counting elements in sets is over counting, consider for finite sets \(A\) and \(B\):

\[n(A \cup B) = n(A) + n(B) - n(A \cap B)\]

which is outlined graphically in Figure Fig. 11.2. We can also state this idea as:

\[n(A \cup B) \leq n(A) + n(B)\]

where the equality holds if the sets are disjoint and the inequality holds otherwise. To avoid this over counting, we can construct the set \(A \cap B\) and so

\[n(A\setminus B) = n(A) - n(A \cap B)\]

We can also extend these statement to many sets, suppose \(A,\,B,\,C\) are finite sets:

\[\begin{split}n(A \cup B \cup C) &= n(A) + n(B) + n(C) \\ &-& n(A\cap B) - n(A\cap C) - n(B \cap C) \\ &+& n(A \cap B \cap C)\end{split}\]

where we need to remove the double counting, but in doing so we loose the triple counting, so we need to re-add this total intersection at the end.
